﻿# 工会经费预决算审核标准参考

> 本文件供 AI 审核时直接引用具体法规条款。如当地工会发布了新版实施细则，以当地最新版本为准。

---

## 一、主要法规依据

### 1. 《工会会计制度》（财会〔2021〕7号）

| 条款 | 核心内容 | 审核应用 |
|------|---------|---------|
| **第十条** | 工会应当对固定资产进行定期清查盘点，每年至少盘点一次 | 检查固定资产是否如实登记，原值不应为0 |
| **第十四条** | 工会经费收支应当按照预算管理的要求，编制年度收支预算和决算 | 检查预决算报表编制是否完整 |
| **第二十八条** | 工会资产、负债、净资产应当按照资产负债表列示。工会资金结余包括工会资金结转和工会资金结余 | **勾稽关系依据**：决算表"本年结余"应与资产负债表"工会资金结余"变动额一致（考虑结转因素） |
| **第三十三条** | 基层工会应当将会费收入、拨缴经费收入等全部收入纳入预算管理 | 检查收入项目是否完整纳入预算 |
| **第四十七条** | 专项补助资金应当专款专用，按照规定的用途和标准使用 | 检查专项补助使用是否合规 |
| **附录——会计科目** | "固定资产"科目核算工会固定资产原值；"累计折旧"核算固定资产折旧 | 检查固定资产净值计算是否正确 |

### 2. 《工会预算管理办法》

| 条款 | 核心内容 | 审核应用 |
|------|---------|---------|
| **第十七条** | 基层工会预算支出包括职工活动支出、维权支出、业务支出、行政支出等 | 检查支出类别是否规范 |
| **第二十五条** | 预算编制应当参考上一年度预算执行情况和本年度收支预测 | 预算数与上年决算数大幅变动应有说明（⚠️ **阈值：变动 > 30%** 应关注） |
| **第三十一条** | 动用上年结余应当控制在当年预算的20%以内，超过20%的需报上级工会批准 | 检查动用上年结余比例是否超过 **20%**，是否有批文 |
| **第三十八条** | 预算执行率偏低或无预算支出的，应在决算中说明原因 | **执行率阈值：< 50%** 应要求说明原因 |
| **第四十二条** | 预算执行中因特殊情况需要调整预算的，应当编制预算调整方案 | 超预算较大（> 20%）应检查是否有调整方案 |
| **第五十条** | 决算草案应当真实、完整、准确反映预算执行结果 | 决算各项数据应有据可查 |

### 3. 《基层工会经费收支管理办法实施细则》

> 各省份文件号不同，以当地总工会发布的最新版本为准。以下为通行规定。

| 项目 | 标准 | 审核要点 |
|------|------|---------|
| **会费收入** | 工资总额 × **0.5%** 逐月收缴 | 检查是否按此比例计算，实收率是否完整 |
| **拨缴经费** | 工资总额 × 2% × 60%（基层留成），再扣除上级工会留成 | 省总工会留成约5%，市总工会约10%，县总工会约15%（以当地为准） |
| **节日慰问品** | 全年不超过拨缴经费收入 | 检查是否超出 |
| **医疗互助** | 应当在预算和决算中体现 | 检查是否有医疗互助支出项 |
| **送温暖支出** | 住院慰问列入送温暖支出，**不得**列入会员活动支出 | 检查科目列支是否正确 |
| **专项补助资金** | 不得用于发放节日慰问品 | 检查资金用途 |
| **采购程序** | 大额采购应有询价/比选/招标记录 | 检查明细表中采购程序是否规范 |

### 4. 上级工会关于专项补助措施的相关文件

（具体文件名称和文号以当地发布为准）

| 要求 | 说明 |
|------|------|
| 专款专用 | 专项补助资金必须按照指定用途使用，不得挪用 |
| 使用范围 | 工会消费券、职工疗休养、文体旅活动、春秋游、技能提升、慰问新标准、采购农产品、两节送温暖 |
| 禁止范围 | 不得用于发放节日慰问品、不得用于日常办公支出、不得用于人员工资 |
| 时效要求 | **当年度专项补助当年度使用完毕，未使用部分可能被上级工会收回** |
| 结余处理 | 未使用完的专项补助资金，应在下年度预算中安排使用计划 |

## 二、量化审核阈值速查

| 指标 | 预警阈值 | 依据 | 审核操作 |
|------|---------|------|---------|
| 固定资产原值 | = 0 | 《工会会计制度》第十条 | 要求全面清查登记 |
| 预算执行率 | < **50%** | 《工会预算管理办法》第三十八条 | 要求说明原因 |
| 拨缴经费超预算 | > **20%** | 《工会预算管理办法》第四十二条 | 检查是否有预算调整方案 |
| 会费收入变动（同比） | 减少 > **30%** | 行业通行标准 | 要求说明原因 |
| 会员活动支出增幅（同比） | > **30%** | 行业通行标准 | 要求说明原因 |
| 动用上年结余 | > **20%** | 《工会预算管理办法》第三十一条 | 检查是否有上级工会批文 |
| 会员活动支出 vs 拨缴经费 | 支出 > 收入 | 《基层工会经费收支管理办法实施细则》 | 超支不予认可 |
| 慰问人数 vs 会员人数 | 不一致 | 《基层工会经费收支管理办法实施细则》 | 要求核实说明 |

## 三、跨表数据勾稽关系检查清单

### 3.1 决算表 ↔ 资产负债表

| 检查项 | 说明 |
|--------|------|
| 决算表"本年结余" | 应与资产负债表"工会资金结余"年末数 - 年初数基本一致 |
| 资产负债表"资产总计" | 应 = "负债合计" + "净资产合计" |

### 3.2 决算表 ↔ 明细表

| 检查项 | 说明 |
|--------|------|
| 决算表"会员活动支出" | 应 = 会员活动支出明细表合计 |
| 决算表"文体活动支出"（如有） | 应与专项补助措施表中文体旅活动支出对应 |

### 3.3 预算表 ↔ 上年决算表

| 检查项 | 说明 |
|--------|------|
| 预算表各项收入/支出 | 与上年决算对应项比较，**变动 > 30%** 应要求说明 |
| 预算表"动用上年结余" | 与资产负债表"工会资金结余"比较，**动用 > 20%** 需批文 |


### 3.4 数据检查表审核要点

数据检查表是预决算报表附带的自动校验表，AI审核时应对照检查：

| 检查项 | 预警值 | 说明 |
|--------|--------|------|
| 动用上年结余比例 | > 20% | 超过需检查是否有上级工会批文 |
| 会员活动支出 / 拨缴经费收入 | > 100% | 当年节日慰问品支出不得超过同期拨缴经费收入 |
| 各支出项目预算执行率 | < 50% | 需在决算中说明原因 |
| 各支出项目超预算比例 | > 20% | 需检查是否有预算调整方案及批复 |

### 3.5 预算表 ↔ 专项补助措施表

| 检查项 | 说明 |
|--------|------|
| 专项补助未使用资金 | 检查是否在下年预算中安排了使用计划 |

### 3.6 专项补助措施表 ↔ 决算表

| 检查项 | 说明 |
|--------|------|
| 已完成的专项补助 | 资金使用应与决算表中对应支出一致（如文体旅活动支出） |

## 四、常见问题编号清单

> AI 发现新问题时，按以下分类和编号体系记录，便于整改复核时追溯。

### 4.1 基本情况表问题

```
问题1 固定资产原值为0或填写不完整
问题2 实收率未填写
问题3 工会干部人数与实际不符
问题4 人员信息填写不规范
问题5 会员人数与职工人数不匹配
```

### 4.2 决算表问题

```
问题1 支出项目说明不完整
问题2 预算执行率偏低（<50%）无说明
问题3 拨缴经费收入大幅超预算（>20%）无说明
问题4 专项资金使用不规范
问题5 数据勾稽关系不一致（决算 ↔ 资产负债表）
问题6 医疗互助支出未体现
问题7 决算表与明细表合计不一致
```

### 4.3 预算表问题

```
问题1 会费收入计算错误（未按0.5%）
问题2 会费收入大幅减少（同比>30%）无说明
问题3 拨缴经费未扣除上级工会留成
问题4 会员活动支出超过拨缴经费收入
问题5 会员活动支出预算增幅过大（同比>30%）无说明
问题6 动用上年结余超过20%无批文
问题7 专项资金使用计划不明确
问题8 医疗互助支出预算未体现
问题9 预算数与上年决算差异大（>30%）无说明
问题10 专项补助未使用资金未在预算中安排
```

### 4.4 会员活动支出明细问题

```
问题1 明细与决算表不一致
问题2 节日慰问品未注明具体节日
问题3 慰问人数与会员人数不符
问题4 住院慰问科目列支错误（应在送温暖支出）
问题5 采购程序填写不规范
问题6 支出与上年决算关联性异常
问题7 支出与专项补助措施不一致
```

### 4.5 专项补助措施问题

```
问题1 专项补助未专款专用
问题2 使用情况与决算表不一致
问题3 覆盖会员人数未填写
问题4 资金使用与上年决算支出关联性不一致
问题5 未使用资金未在下年预算中安排
问题6 当年度专项补助未使用（可能被收回）
```

### 4.6 数据检查表相关问题
- 问题1 动用上年结余超过20%无批文
- 问题2 各支出项目占比违反规定

### 4.7 其他问题（跨表/特殊情况）
- 问题1 跨年欠款列支未在说明中注明
- 问题2 集团工会留成比例未在说明中明确
- 问题3 文体活动支出与专项措施表数据不一致
- 问题4 消费券/专项补助发放与措施表说明内容不一致
- 问题5 慰问金发放填写采购程序为"其他"（慰问金不属于采购范畴）
- 问题6 生日/结婚/生育慰问长期有项目无金额
- 问题7 其他支出（上解经费）错误列入下年预算
- 问题8 账户冻结等不可抗力因素导致拨缴异常，缺少说明
- 问题9 春节慰问品未注明发放的具体节日名称
- 问题10 住院慰问资金来源标注错误（应为送温暖支出）
