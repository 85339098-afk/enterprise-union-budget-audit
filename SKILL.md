## 适用范围

### 适用对象
- 各级企业工会（含国有企业、民营企业、外资企业等各类所有制企业）的年度经费收支预决算审核
- 工会经费审查委员会、上级工会经审部门对下级工会预决算的审核

### 适用场景

> 本节举例以辽宁省总工会相关规定为准，其他省份请据实调整。

| 场景 | 说明 |
|------|------|
| 年度预决算审核 | 对工会年度经费收支决算和预算进行合规性、合理性审核 |
| 整改复核 | 对前期审核意见中列出的问题进行逐项整改情况复核 |
| 公文出具 | 生成符合GB/T 9704-2012标准的正式公文格式审核意见 |
| 七表联审 | 基本情况表、资产负债表、决算表、预算表、专项措施表、会员活动支出明细表、**数据检查表**的联动审核 |

### 不适用的场景

| 场景 | 原因 |
|------|------|
| 机关事业单位工会预决算 | 适用的经费收支管理办法不同，建议使用专门的审核标准 |
| 离任审计 | 此为专项审计，超出本Skill的审核范围 |
| 专项资金审计 | 如需全面审计某项专项资金的使用情况，建议使用专项审计工具 |
| 非年度预决算的其他财务审核 | 本Skill仅针对年度预决算报表设计 |
| 非工会组织的财务审核 | 不适用 |

### 使用前请确认
- [ ] 被审核单位是否为企业工会
- [ ] 本次审核是否为年度经费收支预决算审核
- [ ] 是否已准备好必须提供的审核要件

## 📋 审核工作流程

### 第一步：收集和读取文档

要求用户提供以下文档：
1. **前期审核意见**（docx 格式，如有）
2. **新提交的预决算报表**（xlsx 或 pdf 格式）
3. **参考审核意见模板**（可选，docx 格式）

使用脚本读取：
```bash
python read_budget_docs.py <文件路径>
```

支持的文件格式：
- `.docx` — 审核意见、说明书
- `.xlsx` — 预决算报表（自动读取所有 Sheet）
- `.pdf` — 报表扫描件

### 第二步：前期审核意见整改复核

#### 2.1 复核流程

列出前期审核意见中的所有问题，逐条对比新报表，检查整改情况。

#### 2.2 整改结论处置决策流程

```mermaid
flowchart TD
    A[逐条复核原审核意见] --> B{整改结论判断}
    B -->|已整改| C[记录为"已整改"，无需进一步处理]
    B -->|部分整改| D{原问题性质判断}
    D -->|原问题核心部分未完全解决| E[拆分：<br/>已整改部分 → 写入复核说明<br/>未整改部分 → 写入新问题清单<br/>（标注"部分整改，详见新问题第X条"）]
    D -->|新增了其他问题| E
    B -->|未整改| F[原问题整体 → 写入新问题清单<br/>（标注"未整改，原问题编号"）]
    B -->|需进一步核实| G{能否通过报表数据判断}
    G -->|能| H[根据判断重新归类为已整改/部分整改/未整改]
    G -->|不能| I[记录在案，<br/>在"需核实事项"小节单独列出<br/>要求工会补充说明材料]
    C --> J[生成整改复核对照表]
    E --> J
    F --> J
    I --> J
    J --> K[进入第三步：新问题识别]
```

**核心原则：**
- **已整改**：原问题不再出现在新问题清单中，只在复核对照表中记录
- **部分整改**：已整改部分写在复核说明中，未整改部分作为新问题列出，并在标题中前置标注"〔部分整改〕"
- **未整改**：原问题整体写入新问题清单，标题标注"〔原问题X号未整改〕"
- **需进一步核实**：单独成节"需核实事项"，要求工会限期补充材料

#### 2.3 整改复核对照表格式

| 序号 | 原审核意见 | 整改情况 | 复核结论 | 处置方式 |
|------|-----------|---------|---------|---------|
| 1 | …… | …… | 已整改 | — |
| 2 | …… | …… | 部分整改 | 详见新问题第X条 |
| 3 | …… | …… | 未整改 | 归入新问题清单 |
| 4 | …… | …… | 需进一步核实 | 见"需核实事项"小节 |

### 第三步：新问题识别

参照 `audit_standards.md` 中的审核标准，从六个方面逐表识别新问题。

AI 应按以下结构描述每个问题：

```
- 【涉及报表】决算表 / 基本情况表 / ……
- 【具体位置】第X行 / 第X列 / Sheet名 / 表名
- 【问题描述】具体描述
- 【审核依据】《工会会计制度》第X条 / 《工会预算管理办法》第X条 / ……
- 【整改建议】建议如何整改
```

#### 3.1 基层工会基本情况表审核要点
> 问题编号方式：以下每个“[　]”项对应一个潜在问题，AI审核时应按“问题1、问题2…”连续编号
- [ ] 固定资产原值是否如实登记（不应为0）
- [ ] 实收率是否填写完整
- [ ] 工会干部编制人数是否与实际相符
- [ ] 人员信息是否完整准确

#### 3.2 资产负债表审核要点
- [ ] 年初数与年末数是否平衡
- [ ] 资产总计是否等于负债与净资产总计
- [ ] 固定资产净值计算是否正确

#### 3.3 工会经费收支决算表审核要点
- [ ] 所有支出项目是否有详细说明
- [ ] 用途和人均标准是否明确
- [ ] **预算执行率 < 50%** 是否有说明（参考阈值）
- [ ] 拨缴经费收入超预算 **> 20%** 是否有说明（参考阈值）
- [ ] 专项资金使用是否规范
- [ ] 数据勾稽关系是否正确（本年结余与资产负债表勾稽）
- [ ] 医疗互助支出是否体现
- [ ] 各支出项目备注/说明栏是否填写完整
- [ ] 各表数据间关联性是否有明显异常

#### 3.4 工会经费收支预算表审核要点
- [ ] 会费收入预算是否合理（工资总额 × 0.5%）
- [ ] 会费收入大幅减少（同比下降 **> 30%**）是否有充分说明
- [ ] 拨缴经费收入是否扣除上级工会留成部分（比例依管理层级：省总工会 5%、市总工会 10%、县总工会 15% 等，以当地规定为准）
- [ ] 会员活动支出是否超过拨缴经费收入
- [ ] 会员活动支出预算增幅 **> 30%** 是否有说明
- [ ] 动用上年结余是否超过 **20%**
- [ ] 专项资金使用计划是否明确
- [ ] 医疗互助支出预算是否体现
- [ ] 各收入/支出项目说明栏是否填写完整
- [ ] 预算数与上年决算数的关联性是否合理

#### 3.5 "专项补助措施"落实情况表审核要点
- [ ] 专项补助是否专款专用
- [ ] 使用情况是否与决算表一致
- [ ] 覆盖会员人数是否填写
- [ ] 2025年度专项补助是否使用（**不使用会收回**）
- [ ] 已完成的专项补助：资金使用与上年度决算中的对应支出是否一致
- [ ] 未使用的专项补助：下年度预算中是否安排了使用计划

#### 3.6 会员活动支出明细表审核要点
- [ ] 节日慰问品是否注明具体节日
- [ ] 慰问人数是否准确（与会员人数匹配）
- [ ] 资金来源是否正确（会费 / 经费）
- [ ] 采购程序是否规范（询价/比选/招标）
- [ ] 住院慰问是否列在"送温暖支出"而非"会员活动支出"
- [ ] 明细合计与决算表"会员活动支出"是否一致
- [ ] 各项支出与上年决算对应项目的波动（> 30%）是否有说明
- [ ] 文体旅/春秋游等支出与专项补助措施表对应项目是否一致


#### 3.7 数据检查表审核要点

数据检查表是预决算报表附带的自动校验表，用于检查关键指标是否超出规定范围。审核时应对照以下项目逐项核查：

- [ ] 动用上年结余比例是否 > 20%（如超出需检查是否有上级工会批文）
- [ ] 会员活动支出是否超过拨缴经费收入
- [ ] 各支出项目预算执行率是否 < 50%（需在决算中说明原因）
- [ ] 各支出项目超预算比例是否 > 20%（需检查是否有预算调整方案及批复）
- [ ] 数据检查表提示的其他异常项


### 第四步：形成审核结论

根据整改复核和新问题识别结果，综合形成审核结论：

1. **基本情况概述**：会员人数、职工人数、工资总额
2. **整改复核结果摘要**：已整改 X 项，部分整改 X 项，未整改 X 项
3. **主要问题清单**：按严重程度排序
4. **审核结论**：通过 / 附条件通过 / 退回补正
5. **整改要求**：限期整改（含**5个工作日内报送**要求）

## 审核意见标准用语模板

以下模板从真实审核意见中提炼，AI生成公文时应参照其措辞风格。

### 1. 基本情况

**模板：**
> 【工会名称】（统一社会信用代码：【代码】）填报了2025年度工会经费收支决算及2026年度工会经费收支预算报表。该工会现有会员【人数】人，全部职工【人数】人。

### 2. 整改复核总结

**模板：**
> 针对【前期审核意见】，对你工会整改情况进行复核如下：
> 已整改【X】项，部分整改【X】项，未整改【X】项，需进一步核实【X】项。

### 3. 常见问题标准表述

#### 3.1 固定资产原值为0
> 【描述】基本情况表显示固定资产原值为0。建会时间较长，应有资产需全面清查登记。
> 【依据】《工会会计制度》要求如实登记固定资产。
> 【建议】对固定资产全面清查，如实登记入账。

#### 3.2 专项措施补助未使用
> 【描述】专项措施补助【金额】元当年度未使用。补助应于当年度使用，不得结转。
> 【依据】上级工会关于专项措施的相关文件。
> 【建议】核实使用情况，在下年预算中安排使用计划。

#### 3.3 明细与决算不一致
> 【描述】明细表合计【金额】元，决算表对应科目【金额】元，差额【金额】元。
> 【依据】报表数据应保持一致。
> 【建议】核实差异原因，调整保持一致。

#### 3.4 预算执行率偏低
> 【描述】整体执行率【X%】，具体科目【X%】，执行率偏低。
> 【依据】《工会预算管理办法》要求预算编制科学合理。
> 【建议】说明原因，加强预算执行管理。

#### 3.5 动用结余超20%
> 【描述】本年结余为负值，动用上年结余可能超过20%。
> 【依据】《工会预算管理办法》关于结余资金使用的规定。
> 【建议】核实比例，如超20%需提供上级批文。

#### 3.6 会员活动支出超拨缴经费
> 【描述】会员活动支出预算【金额】元，超过拨缴经费收入【金额】元。
> 【依据】规定：节日慰问品支出不得超过同期拨缴经费收入。
> 【建议】调整预算确保符合规定。

## 🚨 特殊情况处理

### 1. 新成立企业，无上年数据

**处理方法：**
- 基本情况中注明"本年度为新工会首次报送预决算报表，无上年对比数据"
- 预算表审核时，侧重检查编制依据的合理性（如会费收入 = 工资总额 × 0.5% 是否符合）
- 不适用"预算数与上年决算数对比"类审核点
- 决算表审核时，侧重支出合规性，执行率可不做硬性要求

### 2. 企业有多个分会，报表处理方式

**处理方法：**
- 先确认报表是**合并汇总**还是**各分会独立报送**
- 合并汇总的：审核合并数据，同时抽查 1-2 个分会的明细
- 独立报送的：分别审核，在基本情况中注明"本报告涉及 X 个分会"
- 重点关注：分会之间经费划拨是否清晰，是否存在同一项目重复列支

### 3. 报表格式与标准模板不一致

**处理方法：**
- 尝试通过字段名称模糊匹配定位关键数据（如"会费收入""拨缴经费"等）
- 如无法自动匹配，引导用户人工标注对应关系
- 在问题描述中注明"本报表格式与标准模板不一致，以下判断仅供参考"
- 无法读取的数据项标注为"数据缺失，需补充说明"

### 4. 企业规模极小（会员 < 25 人）

**处理方法：**
- 简化审核标准，重点检查：固定资产登记、会费收缴、经费使用合规性
- 不强制要求多级表格勾稽（如可接受无专项补助措施表）
- 结论类标注"鉴于该工会规模较小，审核标准适当放宽"
- 仍保留整改要求中的规范性要求

### 5. 电子数据与纸质扫描件不一致

**处理方法：**
- 优先以纸质件（盖章件）为准
- 在问题描述中注明差异
- 要求工会书面说明差异原因

### 6. 往年欠款列支

**场景：** 部分工会将上一年的费用在当年列支（如2024年的费用在2025年决算中体现）。

**处理方法：**
- 在问题描述中注明含往年欠款的金额及形成原因
- 要求工会在说明栏中单独列出往年欠款部分
- 检查是否在预算说明书中充分披露
- 区分：如果是上年已发生但当年才结算（如跨年合同），可接受但须说明；如果是单纯拖延列支，需整改

### 7. 集团工会留成

**场景：** 集团下属企业工会的拨缴经费需先上缴集团工会，集团工会再上缴上级工会，涉及多层留成。

**处理方法：**
- 先确认集团工会的留成比例（一般为5%，以实际为准）
- 计算公式：拨缴经费收入 = 工资总额 x 2% x 60%（基层留成）- 集团工会留成 - 上级工会留成
- 要求工会在说明栏中明确列明各级留成比例和计算过程
- 如集团内有多级工会，需确认报表报送层级

### 8. 慰问金发放不属采购范畴

**场景：** 会员活动支出明细表中的慰问金发放（住院慰问、去世慰问等）在采购程序栏填写了采购方式。

**处理方法：**
- 慰问金发放属于福利性支出，不属于采购范畴
- 采购程序栏应填写\u201c不适用\u201d或\u201c慰问金发放\u201d
- 实物慰问品采购（如节日慰问品）才需要填写采购程序

### 9. 不可抗力导致的报表异常

**场景：** 企业因账户冻结、政策调整、合并重组等不可抗力原因导致财务数据异常。

**处理方法：**
- 要求工会提供书面说明和相关证明材料
- 在审核意见中单独说明该特殊情况
- 涉及的数据异常标注为\u201c因【原因】导致，待核实\u201d
- 不因特殊情况否定整体审核结论，但需注明

## 🔍 常见问题速查

### Q1: 固定资产原值为什么不应为0？
A: 工会建会时间较长，应有劳模工作室、职工书屋等资产，需全面清查登记。新成立工会除外。

### Q2: 会费收入如何计算？
A: 按工资总额的0.5%逐月足额收缴。**计算公式：会费收入 = 全部职工工资总额 × 0.5% × 12个月 × 实收率**。

### Q3: 拨缴经费收入如何计算？
A: 工资总额 × 2% × 60%（基层工会留成比例），并需扣除上级工会留成部分（比例依管理层级而定）。

### Q4: 会员活动支出有什么限制？
A: 当年发放节日慰问品的活动金额不得超过同期拨缴经费收入。

### Q5: 专项资金可以用于什么？
A: 可用于工会消费券、职工疗休养、文体旅活动、春秋游、技能提升、慰问新标准、采购农产品、两节送温暖。**不得用于发放节日慰问品**。

### Q6: 住院慰问应列入哪个科目？
A: 应列入**送温暖支出**，而非会员活动支出。

### Q7: 数据勾稽关系如何检查？
A: 决算表本年结余应与资产负债表"工会资金结余"变动额一致（考虑工会资金结转因素）。**参考《工会会计制度》第二十八条**。

### Q8: 预算执行率偏低如何判断？
A: 通常以 **< 50%** 为预警线，但需结合实际情况（如项目已取消、政策调整等）。

### Q9: 动用上年结余超过20%会怎样？
A: 根据《工会预算管理办法》第三十一条，**动用上年结余超过20%需报上级工会批准**。审核时重点检查是否有批文。

### Q10: 专项补助资金不使用会怎样？
A: 未在当年度使用的专项补助资金，**上级工会可能收回**。审核时提醒工会尽快使用或申请延期。

---



### Q11: 跨年欠款如何处理？
A: 往年费用在当年列支的，应在说明栏中注明含往年欠款金额。金额较大时应在预算说明书中单独披露。

### Q12: 集团工会留成怎么算？
A: 集团下属企业拨缴经费 = 工资总额 x 2% x 60%（基层留成）- 集团留成 - 上级留成。集团留成比例以实际规定为准。

### Q13: 慰问金需要写采购程序吗？
A: 不需要。慰问金发放属于福利性支出，不属于采购范畴，应填写“不适用”。

### Q14: 本Skill法规依据是全国通用吗？
A: 以**辽宁省**为例编写。各省《基层工会经费收支管理办法实施细则》版本不同，请替换为当地最新版本。

### Q15: 什么是数据检查表？
A: 预决算报表系统自动生成的校验表，检查动用结余比例、支出占比等指标是否超标，审核时应逐项核查。
