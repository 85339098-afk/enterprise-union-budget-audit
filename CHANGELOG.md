## 简介

企业工会预决算审核 Skill 是一款专为企业工会经费预决算审核工作设计的智能辅助工具。适用于各类所有制企业工会的年度经费收支预决算审核。

### 核心能力

| 能力 | 说明 |
|------|------|
| 年度预决算审核 | 对工会年度经费收支决算与预算进行合规性、合理性审核 |
| 整改复核 | 对比前期审核意见，逐项核查整改情况 |
| 七表联审 | 基本情况表、资产负债表、决算表、预算表、专项措施表、会员活动明细表、数据检查表的联动审核 |
| 公文生成 | 生成符合 GB/T 9704-2012 标准的专业审核意见 Word 文档 |
| 法规查询 | 内置《工会会计制度》《工会预算管理办法》《基层工会经费收支管理办法实施细则》核心条款及量化阈值 |
